# kdmytro.com

Personal blog built with [Hugo](https://gohugo.io/) and [PaperMod](https://github.com/adityatelange/hugo-PaperMod), deployed on [Vercel](https://vercel.com/).

## Local Development

```bash
hugo server -D
```

Then open http://localhost:1313

## Deploy

Push to main - Vercel auto-deploys.

## New Post

```bash
hugo new posts/my-post.md
```
